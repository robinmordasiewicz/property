6WIND

6WIND Virtual Accelerator™ is hypervisor acceleration software that works transparently with F5® BIG-IP® Local Traffic Manager™ (LTM®) Virtual Edition (VE) to increase throughput performance without any change to the application, hypervisor, operating system, or management. 6WIND Virtual Accelerator is a software package that accelerates virtual networking infrastructure while reducing CPU resources necessary for packet processing to enable a cost-effective and flexible transition to NFV for Service Provider and Cloud solutions. 6WIND’s testing of F5 BIG-IP LTM VE and Virtual Accelerator under the 6WIND Speed Certification program resulted in a 5x application throughput performance gain running F5 BIG-IP LTM VE on a KVM hypervisor. Together, 6WIND Virtual Accelerator and F5 BIG-IP LTM VE will address performance, feature and flexibility requirements for virtual networking to help our joint customers increase revenue and reduce time to market for new services leveraging NFV and SDN deployment scenarios.

Learn more 

Absolute

The integration of F5 BIG-IP Edge Client for Windows with Absolute® Application Persistence® ensures that the application is running in a healthy state on all endpoints across the device fleet. Application Persistence leverages Absolute's Persistence technology, the only security solution embedded in the firmware of devices. Application Persistence runs regular health checks on F5 BIG-IP Edge Client for Windows' running services, registry files and operational directories and automatically remediates the application in cases of non-compliance. This zero-touch, automated remediation ensures that organizations make the most of their investments in security applications by eradicating the risk of human error and guaranteeing their corporate networks are fully protected at all times.
